# Optimising-Pairwise-Distance-Calculations
This project aims to optimise the pairwise distance calculation files within AMD's AOCL Data Analytics library (dev branch). A detailed report on the changes made and the speedups achieved can be found in "Optimisation Report.pdf". For details on building this repository, see AMD's original repo [here](https://github.com/amd/aocl-data-analytics) as well as AMD's AOCL user guide, API, and other documentation [here](https://www.amd.com/en/developer/aocl.html#documentation). This project was made with AOCL-DA's dev branch from October 2025.
